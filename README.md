# SVC Greetings
Example RESTful JSON mirco-service  

## Getting Started

        gem install bundler

  
1. Install Bundler

        gem install bundler
        bundle install

2. Run tests
  
        bundle exec rspec

3. Run server on port 4567
  
        rackup -p 4567

4. Check `http://localhost:4567/greetings`

5. Check `http://localhost:4567/health_check`
